# Freeserver texture

# 使い方
Actionsからnew releaseを実行すると新しいリリーズが作成されます。

自分でForkを作成して色々編集をして新しくしてPull Requestを出しましょう

jsonファイルにコメントを記述したいときは
```JSON
{
  "__comment": "comment here"
}
```
とするとリリース時に削除してリリースされます
# 注意点
絶っっっったいにbuild.jsを実行しないでください
